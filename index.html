Data Wrangling using Python

The analysis is based on a dataset comprising 113,937 loan records, with each record containing 81 attributes. These attributes encompass information such as loan amount, borrower rate (interest rate), current loan status, borrower income, and more. The dataset spans the years 2009 to 2014. To visualize the data, this project utilizes the matplotlib and seaborn libraries, presenting both univariate and bivariate charts depending on the nature of the variables, whether quantitative or categorical.

SQL Murder Mystery

The SQL Murder Mystery presents an interactive puzzle crafted to both educate and assess SQL database querying skills in an enjoyable and immersive manner. Participants assume the role of a detective tasked with solving a murder case through database querying and data analysis, utilizing Data Querying Language (DQL) extensively.

With a revenue of $128 billion (INR 9.6 trillion) in 2021-22, the companies collectively employ over 935,000 people.
